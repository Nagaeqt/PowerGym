# 💪 PowerGym

Sistema web para gerenciamento de academias, permitindo cadastro de usuários, autenticação, visualização de academias e organização de informações relacionadas ao treino e evolução do aluno.

Projeto acadêmico desenvolvido para a disciplina de **Software Product** do curso de **Análise e Desenvolvimento de Sistemas (ADS)** da **Faculdade Impacta**.

---

# 📌 Objetivo do Projeto

O objetivo do projeto é desenvolver um sistema web utilizando arquitetura em **3 camadas (Front-end, Back-end e Banco de Dados)** para demonstrar conceitos de desenvolvimento de software, organização de código, autenticação de usuários e integração entre interface e servidor.

---

# 🚀 Funcionalidades

- Cadastro de usuários
- Login de usuários
- Autenticação utilizando token
- Consulta de perfil do usuário
- Interface de login moderna
- Estrutura preparada para expansão do sistema
- Base para funcionalidades futuras como:
  - academias próximas
  - acompanhamento de treino
  - planos de academia

---

# 🧩 Arquitetura do Projeto

O sistema foi dividido em **três camadas principais**:

### Front-end
Responsável pela interface do usuário.

### Back-end
Responsável pela lógica da aplicação e API.

### Banco de Dados
Responsável pelo armazenamento das informações do sistema.
